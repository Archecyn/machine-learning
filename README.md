# machine-learning

End-to-end machine learning workflows covering supervised and unsupervised learning, model selection, hyperparameter tuning, and MLOps practices including experiment tracking, containerisation, and cloud deployment.

## Directory Contents

- **ANN**: Artificial Neural Networks - computational model inspired by biological neural networks.
- **ARIMA**: ARIMA - time series forecasting model combining autoregression, differencing, and moving average.
- **association-rules**: Association Rules - data mining technique to discover relationships between items in datasets.
- **CNN**: Convolutional Neural Network - deep learning model for analyzing visual data.
- **datasets**: Sample datasets for machine learning examples.
- **decision-tree**: Decision Tree - predictive modeling tool using flowchart-like structure.
- **gaussian-mixture-models**: Gaussian Mixture Models - probabilistic model representing data as mixture of Gaussian distributions.
- **gradient-boosted-model**: Gradient Boosted Model - ensemble method combining multiple decision trees.
- **hidden-markov-models**: Hidden Markov Models - statistical model for systems with unobserved states.
- **k-means**: K-Means - clustering algorithm grouping data into k distinct clusters.
- **k-nearest-neighbours**: K-Nearest Neighbours - algorithm classifying based on majority class of nearest neighbours.
- **lasso**: LASSO - linear regression with variable selection and regularization.
- **LDA**: Linear Discriminant Analysis - supervised algorithm for classification and dimensionality reduction.
- **linear-programming**: Linear Programming - optimization technique for maximizing profit under constraints.
- **linear-regression**: Linear Regression - statistical modeling for predicting relationships between variables.
- **logitstic-regression**: Logistic Regression - statistical model for predicting categorical outcomes.
- **LSTM**: Long Short-Term Memory - type of RNN for learning long-term dependencies.
- **MCMC**: Markov Chain Monte Carlo - computational algorithm for sampling from complex distributions.
- **naive-bayes**: Naive Bayes - classification algorithm based on Bayes' theorem.
- **PCA**: Principal Component Analysis - dimensionality reduction technique using orthogonal transformation.
- **random-forest**: Random Forest - ensemble learning method using multiple decision trees.
- **RNN**: Recurrent Neural Network - neural network for processing sequential data.
- **SVM**: Support Vector Machines - supervised algorithm for classification and regression.
